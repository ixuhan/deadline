# 任务时间轴

> 项目取名：`deadline`，翻译过来就是截止期限、最终期限，本项目旨在帮助大家在平时的工作中记录自己的任务，同时提醒大家还有多少任务待办，取名 `deadline` 也符合任务时间轴的这个项目的初衷。
